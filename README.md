# Hệ thống Quản lý Bán Sách - Testing

## 🎯 Vai trò
Thực tập sinh Tester (Manual Testing)

## 📌 Mô tả
Dự án thực hiện kiểm thử thủ công cho hệ thống quản lý bán sách nhằm đảm bảo các chức năng hoạt động đúng yêu cầu và phát hiện lỗi trước khi triển khai.

## 🧪 Công việc kiểm thử
- Viết test case cho các chức năng của hệ thống
- Thực hiện test và ghi nhận kết quả
- Báo lỗi (bug report) với mô tả chi tiết

## 🔍 Phạm vi kiểm thử
- Đăng nhập
- Đăng ký
- Đăng xuất

## 🛠 Công cụ sử dụng
- Excel / Google Sheet (test case, bug report)
- GitHub (quản lý source)

## 📂 Tài liệu kiểm thử
- Test case
- Bug report
- Test plan
